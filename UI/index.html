AngularJS-based components

Icons

The icon's text size is dependent of its parent's font size.

Usage

<icon type="icon" prefix="icn" name="generic-cc" size="48" withtext="icon-only" text="Generic credit card"></icon>

<icon type="flag" prefix="fl" name="ca" size="16" text="Canada"></icon>
Attributes
Attribute Type Value
type string icon | flag
prefix string icn (for icons) | fl (for flags)
name string CSS selector given for the icon
size number in string format 16 | 24 | 48 | 64
text string Text that goes with the icon
withtext string (Optional): icon-only - means exclude the text

Tooltip

Usage

This has a tooltip.

<p>This <tooltip title="Lorem ipsum dolor sit amet, consectetur adipiscing elit." text="word"></tooltip> has a tooltip.</p>

What is ?

<p>What is <tooltip title="Hypertext Markup Language" text="HTML"></tooltip>?</p>
Attributes
Attribute Type Value
title string This is the tooltip content, plain text only
text string The word of the tooltip trigger.

Notably (Popover)

Notably can be used to accommodate blocks of information to appear in a bubble. It can contain plain text content (no HTML).

Usage

This is something.

<notably title="This is notably" content="This contains plain text content.">something</notably>

Click the icon .

<notably title="This is notably" content="Lorem ipsum dolor sit amet, consectetur adipiscing elit. This contains plain text content."><icon type="icon" prefix="icn" name="info" size="16" withtext="icon-only" text="Info"></icon></notably>
Attributes
Attributes Type Value
title string This is the header or title of the popover.
content string This is the body copy of the popover.
